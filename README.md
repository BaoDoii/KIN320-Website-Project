The Film Review Website Project
You and 3 other students are building a film review website together. Here's the breakdown:
What the site needs:

A homepage that links to each person's reviewer profile
Each person writes their own film review essay(s) about sports films

Each person's essay can be either:

One long essay (4–5 pages) about a single film, covering multiple points
Multiple shorter reviews (1½–2 pages each) covering 3–4 different films

Important rules:

Double spaced, normal font, written like a professional film magazine
Everyone must use the same citation style (APA, MLA, or ASA)
Show real knowledge of sports films — variety in sport, topic, and style of film
The writing should feel like a real critic, not AI-generated

The website itself:

Hosted for free on Netlify
Graded mostly on the writing, not the design — but it needs to look like you put in effort

Grading is out of 20 points:

Written Content → /6
Critical Analysis → /7
Writing Quality & Citations → /4
Website Effort → /3
